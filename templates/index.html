Data Retrieve

Hello user,

Welcome to the user interface for data retrieve. The intention of this code is to survey curriculum from top MBA programs, specifically with a focus on data analytics. This is the first version, and webscraping resync requires executing code outside of this UI. This will be improved in future versions. All information displayed on this UI is scraped from public curriculum websites, transformed into dataframes using Pandas, loaded into a MongoDB database, and then extracted to run here on HTML using PyMongo and Flask.

I qualified data courses as any courses with the following words in the title: Data, Analytics, Machine Learning, or Compu*. (This last string is to capture "computer" and "computing", etc.) These criteria could be broadened, narrowed, or changed without much trouble.
